db_libs.split_pdb_op.split_pdb(split_pdb, cuttoff_dist=8, min_rec_atom_num=100, min_lig_atom_num=5, discard_h=True, init='split_pdb_init')¶
Iterates through every molecule of the ligand in the receptor. Selects atoms of the receptor (and any chemicals,
but not water) within within the cutoff distance of any atoms of the ligand. Saves ligand + selection of atoms around the receptor’s binding site into the folder.
Parameters:
  • split_pdb – string (relative path the the file to split)
  • cuttoff_dist – float (distance of any atoms in the binding site from any atom of the ligand to be saved)
  • min_rec_atom_num – minimun number of atoms be saved as binding site
  • min_lig_atom_num – minumum number of atoms for ligand to be saved
  • discard_h – Bool T/F (discard all hydrogens for both ligand and receptor defaults to True)
  • init – string (init function in this module)
Returns:

nested list of pairs of file names (multiple ligands each paired with selection of the binding site)
